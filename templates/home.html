Hi there! Welcome to this awesome little web app, which I made after my learnings from numerous Udacity courses. Simply put, this app is a blogging engine which means that you can get your blog posts up on the web in an instant! So check out my blogging app here. And that's not it. Check out the wiki as well. But you may not be able to see your wiki posts right after submitting. I've worked out a solution to it and will fix it soon!

If you're interested, this app was built on Google App Engine with the webapp2 framework. Some of the other technologies and languages used were:

  • Python
  • JavaScript
  • HTML / CSS
  • JQuery
  • Jinja2 templating engine
  • Memcache
  • Web APIs
